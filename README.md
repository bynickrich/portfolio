## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [Bun](https://bun.sh)
- [OGL](https://github.com/oframe/ogl) (WebGL light rays effect via [React Bits](https://www.reactbits.dev/))
- Deployed on [Vercel](https://vercel.com)

## Development

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Deployment

Pushes to `main` trigger automatic production deployments on Vercel.
